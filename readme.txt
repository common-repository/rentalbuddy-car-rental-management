== Installation ==

Install plugin to your wordpress then follow steps below.

1. Add car rental calculator [rentalbuddy_calculator].
2. Add car rental calculator without fleet [rentalbuddy_calculator showcars='NO'].
3. Go to Admin panel RentalBuddy to add manufactures.
4. Go to Admin to add Cars.
